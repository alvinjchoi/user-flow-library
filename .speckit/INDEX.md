## Contributing

When adding a new spec:

1. Copy the appropriate template from `templates/`
2. Create a new file in `specs/` with format `XXX-feature-name.md`
3. Fill out all required sections
4. Add an entry to this index
5. Create a corresponding plan in `plans/` (optional)
6. Create a task list in `tasks/` when ready to implement
7. Link the spec ID in your git commits and PRs
